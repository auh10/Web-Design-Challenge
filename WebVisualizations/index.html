Summary: Latitude vs. X

The purpose of this project was to analyze the how the weather changes as you get closer to the equator. To perform this analysis, we used OpenWeatherMap API to randomly create a dataset of 500 cities.

After compiling the dataset, we used Matplotlib to plot the weather vs. latitude. Factors we explored include: temperature, cloudiness, wind speed, and humidity.

This site provides the source data and visualizations that were created in the analysis, as well as explanations and descriptions of trends and correlations observed.
